多视角全局 3D 态势融合 — 六模块分工

多路无人机视角 → 同一度量坐标系里的全局对象注册表 → 检测 · 跟踪 · counting · 空间关系问答 · 时空预测。全局 3D 靠多视角三角化而非稠密重建。模块之间只经数据契约 + Protocol 通信，效果相互影响、实现相互独立。

契约带版本的 schema 是唯一 API
Protocol算法藏在接口后，换实现=换注入
评测每模块独立 benchmark，改算法只看本模块回归

消费者M5信息压缩 + 检索 语义 / visual-token 嵌入（段级 + 目标级）+ 多视角检索（已建）。 下一步把检索绑到 GlobalObject（目标级命中全局 ID）；检索是初步功能，能用即可。 待定
